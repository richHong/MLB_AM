# MLB AM Take-Home Exercise

###Getting Started
  
  1. Install dependencies:

    ```npm install```

  2. Start server:

    ```npm start```

  3. Go to:

    ```http://localhost:3000```

  4.Controls:

    - Left Arrow Key:  Move Left
    - Right Arrow Key: Move Right
    - Up Arrow Key:    Increase Date
    - Down Arrow Key:  Decrease Date
    - Enter Key:       Toggle Line Score Overlay


###To Run Unit Tests

  ```npm test```
