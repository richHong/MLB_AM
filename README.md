# MLB AM Take-Home Exercise

![Preview](preview.png "Preview")

###Getting Started
  
  1. Install dependencies:

    `npm install`

  2. Start server:

    `npm start`

  3. Go to:

    `http://localhost:3000`

  4. Controls:

| Key         | Action         |
|-------------|----------------|
| Left Arrow  | Move Left      |   
| Right Arrow | Move Right     |
| Up Arrow    | Increase Date  |
| Down Arrow  | Decrease Date  |
| Enter       | Toggle Overlay |


###To Run Unit Tests

  `npm test`
